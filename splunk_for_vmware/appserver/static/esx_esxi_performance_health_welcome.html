The VMware ESX/ESXi Performance Data Health page helps administrators assess the health of their infrastructure.

Checklist:
  1. Check the number of ESX/ESXi hosts reporting performance data.
  2. Check the number of vCenters reporting performance data.
  3. Check the performance types associated with hosts and sources.
  4. Check the data throughput over time.
