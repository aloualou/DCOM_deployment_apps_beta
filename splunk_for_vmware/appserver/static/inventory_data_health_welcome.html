The VMware Inventory Data Health view helps administrators verify that their data is looking healthy and usable.

Checklist:
  1. Number of sources in the database: there should be hundreds of sources.
  2. Forwarder status: Shows each of the forwarder appliances you set up represented here. It does not get affected by the time range picker and is set to 30 minutes.
  3. Number of unique entities seen: the numbers in this panel represent the number of distinct objects in your environment seen in the last 2 hours, therefore the number shown for Virtual Machines should match the number of VM's in your environment.
  4. Counts of entities over time: these bar charts show the number of distinct objects of a particular type, it should remain constant through time, with the exception being when you have added or deleted entities, e.g. cloned a VM
